# La Pâtisserie Academy Website

## Overview

This project is a responsive website for La Pâtisserie Academy, an online baking school specializing in cake baking. The website showcases the academy's 4-week online course, providing information about the course details, testimonials from past students, and a registration form for interested participants.

## Live Demo

You can view the live website here: [La Pâtisserie Academy](https://your-github-username.github.io/la-patisserie-academy)

## Project Structure

The project consists of the following main files:

- `index.html`: The main HTML file containing the structure of the website.
- `styles.css`: The CSS file with all the styles for the website.
- `images/`: A directory containing all the images used in the website.

## Cloning the Repository

To clone this repository to your local machine, follow these steps:

1. Open your terminal or command prompt.
2. Navigate to the directory where you want to clone the repository.
3. Run the following command:
   ```
   git clone https://github.com/your-github-username/la-patisserie-academy.git
   ```
4. Navigate into the cloned directory:
   ```
   cd la-patisserie-academy
   ```

## Running the Website Locally

After cloning the repository, you can run the website locally by opening the `index.html` file in your web browser.
